ℹ UNSW-NB15 Column Reference

            
Key columns used for ML text features

            

              
ColumnDescriptionUsed

              
                
protoNetwork protocol (tcp, udp, …)Yes

                
serviceApplication service (http, ftp, …)Yes

                
stateConnection state (FIN, CON, …)Yes

                
attack_catAttack category (Fuzzers, DoS, …)Yes

                
ct_srv_srcConnections to same service/srcYes

                
ct_state_ttlState/TTL correlation countYes

                
labelBinary label (0=normal, 1=attack)Label

                
dur, sbytes, dbytes…Numeric traffic statsSkip

              
            

            

              Numeric columns are skipped — the TF-IDF vectorizer needs string tokens.
              Download: research.unsw.edu.au/projects/unsw-nb15-dataset
